# Morse Micro Firmware (Upstream Firmware Branch)
This repository contains the binary firmware images and BCFs required for operation of modules based on Morse Micro's WiFi HaLow chips.
*Note: This branch contains the firmware named in the format suitable for use with the upstream mm81x driver. For firmware binaries suitable for use with morse_driver, please check out the branch corresponding to your version.*

## Makefile
The included Makefile will install the standard firmware image and BCFs to `/lib/firmware/morsemicro` for use by the Morse Micro upstream driver.
